# law-check

legal.md에 수록된 법령들의 시행일을 법제처에서 조회하고, 기존 기록과 비교하여 변경 여부를 보고한다.

## 실행 순서

### 1. legal.md 파싱

`domain/legal.md`에서 법령명 목록을 추출한다.

- 각 테이블의 "법령" 컬럼에서 이름 수집
- "확인 시행일" 컬럼에서 기존 기록 확인
- resource/ PDF로만 관리하는 항목(세부사항, 평가매뉴얼)은 조회 대상에서 제외

### 2. 법제처 조회

각 법령에 대해 WebFetch로 법제처(law.go.kr)를 조회한다.

```
URL 패턴: https://www.law.go.kr/법령/{법령명}
```

추출 대상:
- 최종 시행일
- 개정 유형 (일부개정/전부개정)

### 3. 비교 및 보고

| 법령 | 기존 시행일 | 조회 시행일 | 상태 |
|------|:----------:|:----------:|------|
| ... | ... | ... | 변경 없음 / 개정됨 / 신규 확인 |

- **개정됨**: 기존 시행일과 조회 시행일이 다른 경우
- **신규 확인**: 기존 시행일이 "—"(미확인)인 경우
- **변경 없음**: 동일한 경우

### 4. legal.md 업데이트

사용자 확인 후, 변경/신규 항목에 대해:
- "확인 시행일" → 조회된 시행일로 갱신
- "최종 확인일" → 오늘 날짜로 갱신

### 주의사항

- 법제처 페이지 구조가 바뀌면 조회 실패할 수 있음 — 실패 시 수동 확인 안내
- 시행령/시행규칙은 모법과 별도로 개정될 수 있음 — 개별 조회
- 고시는 법제처에서 "행정규칙" 카테고리로 검색
