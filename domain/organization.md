# 케어링 조직 구조

## 본사 (기능조직, F = Function)

```
대표 김태성
├─ 전략이사 김윤지 (대표 직속, 산하 조직 없음)
│
├─ 요양성장이사 정민영
│   ├─ 요양성장F ─ 리드 정민영(기획) / 리드 김두영(지원)
│   │   ├─ 경영기획팀
│   │   ├─ 성장기획팀
│   │   ├─ 성장지원팀
│   │   ├─ 마케팅팀
│   │   └─ 고객컨택팀
│   │
│   └─ 요양시스템F ─ 리드 유현석(PO) / 리드 정은구(개발) ← 사용자 소속
│       ├─ 시스템기획팀
│       └─ 시스템개발팀
│
├─ 통합요양행정F ─ 리드 이군도
│   ├─ 행정기획팀
│   └─ 행정운영팀
│
├─ 요양심사F ─ 이사 성현진
│   ├─ 주간보호심사팀
│   ├─ 통합요양심사팀
│   └─ 대구 서구점(주보) ← 본사 주도 테스트베드 센터 (성장 전략 빠른 적용·검증 목적)
│
├─ 인사이사 박수진
│   ├─ 피플F ─ 리드 유단비
│   │   ├─ 인사팀
│   │   └─ 채용팀
│   │
│   └─ 총무F ─ 리드 이동원(총무) / 리드 강태동(공간기획)
│       ├─ 총무팀
│       └─ 공간기획팀
│
└─ 재무이사 박대성
    └─ 재무F
        └─ 재무팀
```

## 본부 (목적조직, 서비스 운영)

본부는 실제 장기요양 서비스를 제공하는 운영 본체.

### 본부 목록 (6개)

수도권1본부 · 수도권2본부 · 수도권3본부 · 영남본부 · 충청본부 · 호남본부

### 본부 내부 구조

```
본부장
├─ 본부관리팀
├─ 주간보호센터 ─ 센터장 → 사회복지사
└─ 통합요양센터 ─ 센터장 → 사회복지사
```

- **통합요양센터** = 방문요양·방문목욕·방문간호를 한 센터에서 운영
- **수급자**(서비스 수혜)와 **요양보호사**(서비스 제공)는 조직도 외부의 **고객**으로 분류
- 본사·본부 모두 운영 모니터링 목적으로 ERP/CAATS 사용
