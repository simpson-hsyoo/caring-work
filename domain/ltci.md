# 장기요양보험 제도

## 개요

- **근거 법령**: 노인장기요양보험법 (2008년 시행)
- **운영 주체**: 국민건강보험공단 (이하 공단)
- **목적**: 고령·노인성 질병으로 일상생활이 어려운 국민에게 장기요양급여를 제공하여 가족의 부담을 덜고 국민의 삶의 질 향상 (법 제1조)
- **재원**: 장기요양보험료 (건강보험료의 일정 비율) + 국가·지자체 부담 + 본인부담금

---

## 장기요양인정 절차

```
신청 → 방문조사 → 등급판정위원회 심의 → 장기요양인정서 발급 → 서비스 이용
```

| 단계 | 주체 | 내용 |
|------|------|------|
| 신청 | 수급 희망자(또는 대리인) | 공단에 장기요양인정 신청 |
| 방문조사 | 공단 직원 | 심신상태 조사 (52개 항목), 장기요양인정점수 산출 |
| 등급판정 | 등급판정위원회 | 인정점수 + 의사소견서 기반 등급 판정 |
| 인정서 발급 | 공단 | 등급, 유효기간, 급여 종류·한도 기재 |
| 서비스 이용 | 수급자 | 장기요양기관 선택 → 급여 이용 개시 |

- 인정 유효기간: 등급·갱신 여부에 따라 다름 → [고시 세부사항](../resource/장기요양급여%20제공기준%20및%20급여비용%20산정방법%20등에%20관한%20세부사항%20전문.pdf) 참조
- 갱신·등급변경 신청 가능

---

## 등급 체계

| 등급 | 인정점수 | 상태 요약 |
|------|----------|-----------|
| 1등급 | 95점 이상 | 일상생활 전적 의존 |
| 2등급 | 75점 이상 ~ 95점 미만 | 상당 부분 타인 도움 필요 |
| 3등급 | 60점 이상 ~ 75점 미만 | 부분적 도움 필요 |
| 4등급 | 51점 이상 ~ 60점 미만 | 일정 부분 도움 필요 |
| 5등급 | 45점 이상 ~ 51점 미만 | 치매 수급자 (경증) |
| 인지지원등급 | 45점 미만 + 치매 | 치매 진단이나 신체기능 양호 |

- 등급이 낮을수록(숫자가 클수록) 경증
- 케어링은 고시 규정 등급 체계를 그대로 따름 (별도 제한 없음)

## 급여 유형

### 재가급여

| 급여 | 내용 | 케어링 제공 |
|------|------|:-----------:|
| 방문요양 | 요양보호사가 가정 방문, 신체·가사 활동 지원 | O |
| 방문목욕 | 목욕 장비 갖춘 차량으로 방문, 목욕 서비스 | O |
| 방문간호 | 간호사 등이 방문, 간호·진료 보조·상담 | O |
| 주야간보호 | 주간(또는 야간) 시설에서 심신기능 유지·향상 프로그램 | O |
| 단기보호 | 시설에 단기 입소 (가족 부재 시 등) | X |
| 복지용구 | 요양에 필요한 용구 구입·대여 | X |

- **통합요양** = 방문요양 + 방문목욕 + 방문간호 (케어링 용어)

### 시설급여

| 급여 | 내용 | 케어링 제공 |
|------|------|:-----------:|
| 노인요양시설 | 입소 정원 10인 이상 | X |
| 노인요양공동생활가정 | 입소 정원 5~9인 | X |

- 케어링은 **재가급여 중심** 사업자

## 수가·청구 구조

### 월 한도액

등급별로 공단이 정한 월 이용 한도액이 있으며, 한도 내에서 급여를 조합하여 이용.

| 등급 | 월 한도액 |
|------|-----------|
- 등급별 금액은 매년 고시로 개정됨 → [고시 세부사항](../resource/장기요양급여%20제공기준%20및%20급여비용%20산정방법%20등에%20관한%20세부사항%20전문.pdf) 참조
- 한도 초과분은 전액 본인 부담

### 수가 산정

| 급여 유형 | 산정 단위 | 비고 |
|-----------|-----------|------|
| 방문요양 | 시간당 | 30분·60분·90분·120분·150분·180분·210분·240분 구간별 |
| 방문목욕 | 회당 | 차량 이용 여부에 따라 차등 |
| 방문간호 | 회당 (시간 구간별) | 30분 미만·30~60분·60분 이상 |
| 주야간보호 | 일당 (시간 구간별) | 3시간·6시간·8시간·10시간·12시간 등 |

### 본인부담금

| 구분 | 본인부담 비율 |
|------|:------------:|
| 재가급여 | 15% |
| 시설급여 | 20% |
| 기초생활수급자 | 면제 |
| 의료급여 수급자 등 | 감경 — 노장법 시행령 참조 |

### 청구 흐름

```
급여 제공 → 급여 제공 기록 작성 → 공단에 비용 청구 → 공단 심사 → 지급
```

- 장기요양기관(= 케어링 센터)이 공단에 월 단위로 청구
- 공단포털(노인장기요양정보시스템)을 통해 전자 청구

## 케어링 관점 핵심 포인트

- 케어링은 **재가급여 4종**(방문요양·방문목욕·방문간호·주야간보호)을 제공하는 장기요양기관
- 매출 = Σ(등급별 수가 × 이용량) - 한도액 제약
- 데이터 최신성이 공단포털에 종속 → 인정서 정보·등급 변경·급여 실적 등을 공단에서 수집
- 급여 제공 기록은 ERP에서 관리하며, 청구 데이터와 연동
