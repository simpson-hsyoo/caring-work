# 장기요양기관 평가 체계

## 개요

- **근거 법령**: 노인장기요양보험법 제54조(장기요양급여의 관리·평가)
- **세부 규정**: 같은 법 시행규칙 제38조, 장기요양기관 평가방법 등에 관한 고시, 장기요양기관 평가관리 시행세칙
- **목적**: 장기요양 서비스의 질 향상 + 국민의 기관 선택권 보장
- **운영 주체**: 국민건강보험공단

---

## 평가 주기·대상

| 항목 | 내용 |
|------|------|
| 주기 | 정기평가 3년 주기 (노장법 시행규칙 제38조) |
| 대상 | 장기요양기관으로 지정된 모든 기관 (급여 유형별 개별 평가) |
| 거부 시 | 최하위 등급 조정 + 과태료 부과 (법 제69조) + 행정처분 + 명단 공표 |

- 케어링은 방문요양·방문목욕·방문간호·주야간보호 4개 급여 유형에 대해 **각각** 평가를 받음

---

## 평가 절차

```
평가예정통보 (7일 전) → 기관 방문 → 기록·현장·면담·시연·전산·유선 확인
→ 전자평가조사표 전송 → 상담·총평 → 기관 확인·동의 → 평가위원회 심의 → 결과 공표
```

### 평가 방법 6가지

| 약칭 | 내용 |
|------|------|
| 기록 | 기관의 문서·자료 등 기록 확인 |
| 현장 | 기관 내·외부 환경 확인 |
| 면담 | 직원·수급자 대상 면담 |
| 시연 | 직원·수급자 대상 시연 확인 |
| 전산 | 공단 정보시스템 전산 자료 확인 |
| 유선 | 수급자(보호자) 전화 확인 |

## 평가 척도·점수

### 지표별 척도

| 척도 | 배점 비율 |
|------|-----------|
| 우수 | 해당 지표 점수의 100% |
| 양호 | 75% |
| 보통 | 50% |
| 미흡 | 0% |
| 해당없음 | 총점 산출에서 제외 |

### 세부점수 평점기준

| 세부점수 | 기준 |
|----------|------|
| 1 | 평가기준을 모두 충족 |
| 0.75 | 대부분 충족 |
| 0.5 | 일부 충족 |
| 0 | 충족하지 못함 |

- 면담·시연 항목: 표본별 세부점수 평균 ≥ 0.75 → "충족"
- 최종 총점은 **100점 환산** (해당없음 지표 제외 후 환산)

### 평가 결과 등급·가감지급

- 평가 결과는 등급으로 산정되며, 등급에 따라 **급여비용 가감지급** 적용 (시행규칙 제31조의2)
- 상위 등급: 가산금 → 직원 처우개선에 사용 의무
- 하위 등급: 감산
- 등급 분류 기준 및 가감 비율 → [장기요양기관 평가방법 등에 관한 고시](../domain/legal.md) 참조

## 평가 영역·지표 구조

### 공통 3대 평가 영역

모든 급여 유형이 **기관운영 → 급여제공과정 → 급여제공결과** 3개 대분류를 공유하며, 중분류·소분류·지표 수는 급여 유형별로 다름.

### 급여 유형별 지표 개요

| 급여 유형 | 지표 수 | 특화 지표 (다른 유형에 없는 것) |
|-----------|:-------:|-------------------------------|
| 방문요양 | 34 | 수급자 낙상예방, 인지돌봄, 신체청결상태 |
| 방문목욕 | 34 | 목욕장비 관리, 목욕 시 욕조 소독 |
| 방문간호 | 34 | 기구소독관리, 비품구비, 간호기록 |
| 주야간보호 | 48 | 시설기준, 낙상예방환경, 소방시설, 프로그램(신체·인지·사회적응), 급식영양, 투약관리, 이동서비스 |

- 주야간보호는 시설 기반이라 시설·프로그램·급식 관련 지표가 추가되어 지표 수가 많음

### 공통 대분류 → 중분류 구조

```
기관운영
├── 기관관리 (운영원칙·체계, 인력운영, 직원 후생복지)
├── 환경 및 안전 (정보관리, 위생·감염관리, 안전관리)
├── 수급자 권리보장 (수급자 존엄성, 수급자의 알권리)
└── 기관책임 (상담관리, 관리자의무, 윤리적운영)

급여제공과정
├── 급여개시 (정보제공, 계약체결·통보, 욕구사정)
└── 급여제공 (개별적 욕구반영, 급여제공, 안전관리, 기능향상)

급여제공결과
├── 사례관리 회의
├── 결과평가
├── 수급자관리 (수급자 상태, 등급현황)
└── 만족도평가
```

## 케어링 관점 핵심 포인트

- 4개 급여 유형 × 각각 평가 = **센터당 최대 4건의 평가** 대응 필요
- 평가 등급이 급여비용 가감에 직결 → **매출에 직접 영향**
- 주야간보호가 지표 48개로 가장 복잡 (시설·프로그램·급식 등)
- 공통적으로 비중이 큰 영역: 직원 교육·인권, 욕구사정·급여제공계획, 노인인권보호
- ERP에서 관리하는 데이터(급여제공기록, 상담기록, 욕구사정 등)가 평가 근거 자료의 상당 부분을 차지
