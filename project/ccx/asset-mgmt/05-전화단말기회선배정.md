# 5. 전화 단말기·회선 배정

## 시나리오 정의

| 항목 | 내용 |
|------|------|
| 트리거 | 배정 필요 |
| 행위자 | 배정자 (총무F) |
| 입력 | 대상 owner, 전화 단말기, 회선, 사용 위치 |
| 출력 | 전화 단말기↔회선↔owner 매핑<br/>+ 자산·회선 상태 → 활성<br/>+ 이력 |
| 사전조건 | 전화 단말기·회선이 사용 가능 상태 |
| 사후조건 | 전화 단말기↔회선↔owner 매핑 활성 |
| 연관 카테고리 | [1](01-신규자산취득.md) / [3](03-신규회선등록.md) (배정 전 등록), [6](06-전화단말기회선회수.md) (회수) |

## Step 시퀀스

| # | 행위자 | 행위 | 분기/예외 |
|---|--------|------|-----------|
| 1 | 배정자 | 배정 트리거 진입 | — |
| 2 | 배정자 | 대상 owner 선택 | 사용자 / 조직 공용 |
| 3 | 배정자 | 전화 단말기 선택 | — |
| 4 | 배정자 | 회선 선택 | — |
| 5 | 배정자 | 사용 위치 지정 | — |
| 6 | 배정자 | 부가서비스 설정 | Y / N |
| 7 | 시스템 | 전화 단말기↔회선↔owner 매핑 생성<br/>+ 상태 변경<br/>+ 이력 기록 | — |

## Mermaid Flowchart

```mermaid
flowchart TD
    Start([전화 단말기·회선 배정 시작]) --> Trigger[배정 트리거 진입]
    Trigger --> Subject{대상 owner 종류}
    Subject -->|사용자| User[사용자 선택]
    Subject -->|조직 공용| Org[조직 단위 선택]
    User --> Device
    Org --> Device
    Device[전화 단말기 선택] --> Line[회선 선택]
    Line --> Loc[사용 위치 지정]
    Loc --> SvcOpt{부가서비스 설정 여부}
    SvcOpt -->|Y| AddSvc[부가서비스 선택]
    SvcOpt -->|N| Mapping
    AddSvc --> Mapping[전화 단말기↔회선↔owner 매핑 생성<br/>+ 상태 변경<br/>+ 이력 기록]
    Mapping --> End([배정 완료])
```
